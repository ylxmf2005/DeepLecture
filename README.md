# DeepLecture

> **开源 AI 视频学习平台——从「听懂」到「学会」，一站式自部署。**

DeepLecture 不只是字幕翻译工具，也不只是 AI 笔记生成器——它是唯一同时覆盖**视频字幕翻译**和**学习资料生成**的开源平台，所有数据完全在本地。

## Highlights

🎙️ **AI 配音 & 自由切换** — 用你喜欢的声音和语言重新配音。智能音频对齐：老师啰嗦时自动加速画面，确保配音和视频始终同步。一个视频可以生成多条配音轨道，播放时一键切换，离开标签页自动切回原音。

📖 **PDF → 讲座视频** — 上传幻灯片，AI 逐页生成讲解脚本并配音，自动合成带字幕的完整讲座视频。支持跨语言：中文课件生成英文讲解。

⏭️ **智能跳过 & 专注模式** — AI 识别闲聊、重复内容并自动快进，只在重点处正常播放。离开标签页自动暂停，回来后告诉你错过了什么。

🔤 **字幕增强** — 不是逐句直译。AI 先提取全文术语表和主题上下文，增强 ASR 原始字幕（修正识别错误、改善断句），再基于上下文翻译，效果远超普通机翻。

📸 **截图即解释** — 看到不懂的图表/公式，截图后 AI 结合前后字幕上下文帮你解释，不用切出去问 ChatGPT。

📚 **一站式学习资料** — 闪卡、Quiz、试卷、Cheatsheet、播客、AI 笔记、事实核查——全部从视频内容一键生成，全部在同一个平台。

## 功能一览

### 🎬 看懂视频
- **双语字幕** — Whisper 转录 + LLM 上下文增强翻译，点击字幕跳转
- **时间线知识点** — AI 按语义拆分知识点（非简单时间切割），每个节点附详细解释
- **智能跳过** — 基于时间线自动快进废话段落
- **专注模式** — 离开自动暂停，回来弹窗总结错过的内容，可一键跳回/提问/加笔记
- **截图解释** — 截图 + 周围字幕上下文 → AI 即时解释，自动保存
- **内置词典** — 悬停/点击字幕查词，发音、释义、一键收藏生词本

### 🎙️ 换个声音听
- **AI 配音** — 多语言/多声音配音，智能音频对齐（画面自动变速）
- **多轨管理** — 一个视频多条配音轨道，播放时自由切换，离开自动切回原音
- **课件生成视频** — PDF 幻灯片 → AI 讲解脚本 + 配音 → 带字幕的完整讲座视频
- **视频导出** — 下载时自选音轨（原音/配音）+ 字幕（单语/双语/不烧录）

### 📚 学习巩固
- **AI 问答** — 一键提问，自动携带时间线/字幕/截图上下文，多轮对话
- **笔记** — WYSIWYG 编辑器 + KaTeX 公式，边看边记，也可 AI 一键生成
- **书签** — 视频任意位置打标记，进度条直接显示
- **闪卡** — AI 生成，按重要性（高/中/低）过滤，区分 STEM / 人文
- **Quiz** — AI 生成选择题，可控题量、难度、关注方向
- **试卷** — 完整考试级试卷生成
- **Cheatsheet** — 1-10 页浓缩速查表，可编辑
- **事实核查** — AI 验证讲座中的声明，生成核查报告

### 🎧 听着学
- **播客** — 讲座转双人对谈播客（主持人 + 嘉宾），可分别设置声音
- **朗读笔记** — 流式 TTS 逐句朗读，实时高亮当前句子
- **Live2D 伴学** — 屏幕上 Live2D 角色，嘴型与音频同步

### ⚙️ 个性化
- **Learner Profile** — 设置学习者画像，影响所有 AI 输出的深度和风格
- **项目管理** — 视频按项目分组，自定义颜色和图标
- **层级配置** — 全局 → 内容 → 任务三层覆盖，不同任务可用不同 LLM/TTS 模型

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- [uv](https://docs.astral.sh/uv/)（Python 包管理器）
- FFmpeg（用于音视频处理）
- Git

### 1. 克隆仓库

```bash
git clone https://github.com/your-username/DeepLecture.git
cd DeepLecture
```

### 2. 安装依赖

**后端**：
```bash
uv sync

# Windows 用户需要额外安装 faster-whisper
uv sync --extra faster-whisper
```

**前端**：
```bash
cd frontend && npm install && cd ..
```

### 3. 配置

复制默认配置文件：

```bash
cp config/conf.default.yaml config/conf.yaml
```

编辑 `config/conf.yaml`，参考注释配置（LLM 为必须配置项）。

### 4. 启动服务

**启动后端**（端口 11393）：
```bash
uv run deeplecture
```

**启动前端**（新终端，端口 3000）：
```bash
cd frontend && npm run dev
```

访问 http://localhost:3000 开始使用。

> **注意**：首次使用字幕功能时，系统会自动下载 Whisper 模型（large-v3-turbo 约 1.5GB），请耐心等待。
