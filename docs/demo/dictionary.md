# 词典与生词本

DeepLecture 内置词典功能，支持在字幕中选词查询释义、听发音、收藏生词，并提供独立的生词本页面方便复习。

## 功能概览

- **选词查词** - 悬浮或点击字幕中的单词即可查看释义
- **发音播放** - 点击音频按钮听标准发音
- **生词收藏** - 一键收藏到生词本
- **生词本页面** - 在 Header 中进入，查看所有收藏的词汇

## 选词查词

### 悬浮模式（默认）

将鼠标悬浮在字幕中的单词上，即可自动弹出词典卡片，显示：

- 单词原形
- 音标
- 释义（多个词性分别列出）
- 例句

鼠标移开后卡片自动关闭。

### 点击模式

如果你觉得悬浮触发过于敏感，可以在设置中切换为「点击模式」：

1. 打开设置（齿轮图标）
2. 进入「功能」标签页
3. 找到「词典」部分
4. 将「交互模式」切换为「点击」

在点击模式下：
- 单击单词弹出词典卡片
- 单击不会触发字幕跳转（视频不会跳到该时间点）
- 点击卡片外部或按 Esc 关闭卡片

### 关闭词典

如果你不需要词典功能，可以完全关闭：

1. 打开设置
2. 进入「功能」标签页
3. 关闭「启用词典」开关

## 发音播放

词典卡片中如果有发音资源，会显示一个喇叭图标：

- 点击喇叭图标播放发音
- 发音来源于 Free Dictionary API
- 部分生僻词可能没有发音，此时不显示喇叭图标

> **注意**：发音需要网络连接。如果播放失败，图标会显示为禁用状态。

## 生词收藏

在词典卡片中点击「收藏」按钮，即可将单词保存到生词本。收藏的信息包括：

- 单词和音标
- 释义
- 发音链接（如有）
- 来源上下文（当前字幕句子）
- 收藏时间

已收藏的单词会显示「已收藏」状态，再次点击可以取消收藏。

## 生词本页面

点击页面顶部 Header 中的「Vocabulary」链接，进入生词本页面：

- 查看所有收藏的单词
- 按视频或全部显示
- 点击单词可以播放发音
- 点击上下文可跳转到对应视频时间点
- 点击删除按钮移除单词

## 设置选项

在「设置 → 功能」中可以调整以下选项：

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 启用词典 | 开启/关闭词典功能 | 开启 |
| 交互模式 | 悬浮触发或点击触发 | 悬浮 |

## 支持的语言

目前词典功能仅支持英语（English）。其他语言的字幕中，词典功能会自动禁用。

## 数据存储

- 生词数据保存在浏览器本地存储（localStorage）
- 清除浏览器数据会丢失收藏的生词
- 建议定期导出重要的生词（未来功能）
