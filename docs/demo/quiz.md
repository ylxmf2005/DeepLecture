# Quiz 小测验

AI 根据课程内容自动生成选择题，用于复习与自测。

## 功能特点

- **智能出题**：从课程字幕/讲义中提取知识点，生成选择题
- **误解多样性**：每个干扰项针对不同类型的常见错误
- **详细解析**：每道题附带正确答案说明和错误选项分析
- **知识分类**：支持公式、定义、条件、算法、常量、示例等类型

## 使用场景

1. **课后复习**：学完一节课后，生成小测验检验理解
2. **考前自测**：快速回顾重点知识
3. **查漏补缺**：通过错误选项分析发现知识盲点
