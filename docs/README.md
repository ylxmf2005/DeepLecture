# DeepLecture 功能演示

欢迎来到 DeepLecture 功能演示页面！这里详细介绍了每个功能的使用方法和演示视频。

## 📚 核心功能

### 字幕与翻译
- **[双语字幕](features/dual-subtitle.md)** - 自动转录并翻译，双语对照学习
- **[AI 配音](features/ai-voiceover.md)** ⭐ - 用你喜欢的声音重新配音

### 内容理解与导航
- **[时间线节点](features/timeline.md)** - AI 自动拆分知识点，可视化课程结构
- **[截图解释](features/screenshot-explanation.md)** - 一键截图，AI 结合上下文解释
- **[AI 问答](features/ai-qa.md)** - 随时提问，自动结合上下文回答

### 智能播放
- **[智能跳过](features/smart-skip.md)** - 自动识别并加速低价值内容
- **[专注模式](features/focus-mode.md)** - 离开自动暂停，回来显示摘要

### 笔记与学习
- **[笔记](features/notes.md)** - WYSIWYG + KaTeX，自动关联视频时间点
- **[Quiz 小测验](demo/quiz.md)** - AI 根据课程内容生成选择题
- **[词典与生词本](demo/dictionary.md)** - 选词查词、发音、收藏与复习

### 高级功能
- **[课件生成视频](features/slide-lecture.md)** ⭐ - 上传 PDF，AI 生成讲解和配音
- **[Live2D 陪伴](features/live2d.md)** - 虚拟形象陪伴，嘴型同步

## 💡 使用技巧

1. **首次使用**：建议先启用双语字幕和时间线节点，快速了解课程内容
2. **深度学习**：遇到难点时使用截图解释和 AI 问答，同时记笔记
3. **效率优先**：开启智能跳过和专注模式，节省时间
4. **外语课程**：使用 AI 配音 + 双语字幕，降低听力负担

## 🚀 进阶玩法

- **组合使用**：时间线节点 + 智能跳过 = 精准定位 + 自动加速
- **复习模式**：笔记 + 时间点跳转 = 快速回顾关键内容
- **沉浸体验**：AI 配音 + Live2D = 流畅舒适的学习体验
